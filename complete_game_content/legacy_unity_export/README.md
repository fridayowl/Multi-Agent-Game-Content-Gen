   - Import all assets

## Troubleshooting

### Common Issues

**NPCs not moving**
- Check patrol points are assigned
- Verify NPCController script is enabled

**Quests not starting**
- Verify QuestManager is in scene
- Check NPC dialogue is set up

**UI not responding**
- Check UIManager references
- Verify Canvas and EventSystem exist
